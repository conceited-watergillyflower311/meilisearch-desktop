## 📦 What is meilisearch-desktop?

Meilisearch Desktop is a simple program that helps you manage Meilisearch search engines on your computer. It works on Windows and other platforms, so you don't need to set up a server or use complicated tools. You get a native desktop app with your data saved right on your computer.

The app blends two technologies to work smoothly:

- **Rust backend**: This runs the core part and keeps your data safe on your machine.
- **React frontend**: This gives you a clean and easy user interface to interact with Meilisearch.

You do not need to know how to use programming tools or command lines. The software gives you buttons and forms to handle everything.

## 📋 Common Tasks Explained

### Adding a New Meilisearch Instance

1. Open the app.
2. Click **Add Instance**.
3. Enter the name and the connection details of your Meilisearch server. If you don't have one, the app can create a local instance for you.
4. Click **Save**.

### Creating Collections (Indexes)

1. Choose your Meilisearch instance.
2. Click **New Index**.
3. Name your index.
4. Start adding documents or data files.

### Searching Data

1. Select an index.
2. Enter your search term in the search bar.
3. Results appear immediately below.
